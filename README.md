# Kernel Principal Component Analysis
Kernel PCA gives us an ability to explore non-linear, complex dataset to unearth patterns that are inconceivable to Linear PCA. Kernel PCA maps featuresets, indirectly (using the [kernel trick](https://www.cs.cornell.edu/courses/cs4780/2018fa/lectures/lecturenote13.html)), to higher dimensions. In essence it detangles distinctive features into dimensions where they can be analysed without the perturbations of unrelated features.
